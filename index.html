dFund Info

A webapp for viewing stats of decentralized investment funds running on the Ethereum blockchain.

  • Uses multiple APIs to gather a unique look into the burgeoning decentralized fund ecosystem.
  • Asynchronously fetches, memoizes, and caches blockchain and cryptocurrency price data
  • Sort and filter fund data in a responsive table
  • Includes a framework for adding new fund platforms
  • Technologies include HTML5, SCSS, jQuery, Webpack, Web3.js, and GraphQL
